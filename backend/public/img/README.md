# Imágenes del Sistema

Este directorio contiene las imágenes utilizadas en el sistema.

## Imágenes requeridas:

### Logo de Romero Panificados
- **Archivo:** `logo-romero.png`
- **Ubicación:** Coloca aquí la imagen del logo de Romero Panificados
- **Tamaño recomendado:** Ancho máximo 400px, altura proporcional
- **Formato:** PNG con fondo transparente (recomendado)

### Escudo de la Escuela Técnica N°1 de Brandsen
- **Archivo:** `escudo-tecnica1.png`
- **Ubicación:** Coloca aquí la imagen del escudo de la Escuela Técnica N°1 Ing. José Rafael Cantón
- **Tamaño recomendado:** 200x200px o similar
- **Formato:** PNG con fondo transparente (recomendado)

## Instrucciones:

1. Coloca el logo de Romero Panificados como `logo-romero.png`
2. Busca y descarga el escudo de la Escuela Técnica N°1 de Brandsen y guárdalo como `escudo-tecnica1.png`

Si no tienes las imágenes, puedes:
- Contactar a Romero Panificados para obtener el logo oficial
- Visitar el sitio web de la Escuela Técnica N°1 de Brandsen para obtener el escudo
- Usar imágenes temporales mientras obtienes las oficiales
